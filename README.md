## Установка

```bash
$ npm install
```

## Создание .env
Создайте .env файл (для образца дан env.sample)

```bash
#Адрес casper-back
NEXT_PUBLIC_API_HOST=http://localhost:3010
```

## Запуск

После запуска casper-back и casper-processor нужно запустить casper-front

```bash
$ npm run build
$ npm run start
```
